# 📌 Project 1: Basic Calculator Program

## 📖 Description
This project is a simple calculator developed using C programming language.  
It performs:

- Addition
- Subtraction
- Multiplication
- Division

using switch case.

---

## 🚀 Features
- Menu Driven Program
- Arithmetic Operations
- User Input Handling
- Switch Case Implementation

---

## ▶️ How to Run

Compile:
```bash
gcc Calculator.c -o calculator
```

Run:
```bash
calculator.exe
```
